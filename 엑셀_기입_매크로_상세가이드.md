# 엑셀 기입 매크로 상세가이드

> 통계분석 자료 전처리 최종 파일을 활용하여, 회원관리시스템(CMS)과 엑셀을 오가며 자동 기입하는 운영 가이드입니다.

---

## 문서 목적

- 운영자가 **반복 입력 업무를 안전하게 자동화**할 수 있도록 실행 절차를 표준화합니다.
- PC 환경별 차이(좌표, 권한, 창 배치)로 발생하는 오류를 최소화합니다.

---

## 사전 준비

### 1) 필수 환경

- OS: Windows 권장
- Python 3.11+
- Visual Studio Code
- Excel
- 회원관리시스템(CMS)

### 2) 파일/폴더 구조(중요)

- 프로젝트 루트에서 실행
- 입력 폴더: `input/data_preprocessing/`
- 출력 폴더: `output/data_preprocessing/`
- 현재 코드 기준 전처리 입력 파일명(고정):
  - `제98기 일일접수세부내역(20251215-20260222).csv`

---

## STEP 1. Visual Studio Code 관리자 권한 실행

1. VS Code 아이콘 우클릭
2. **관리자 권한으로 실행** 클릭

최초 1회 PowerShell 실행 정책 설정이 필요할 수 있습니다.

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

---

## STEP 2. 가상환경/라이브러리 설치 및 실행

아래 명령은 **프로젝트 루트 경로**에서 실행합니다.

1) 가상환경 생성

```powershell
python -m venv .venv
```

2) 가상환경 활성화

```powershell
.\.venv\Scripts\Activate.ps1
```

3) 필수 라이브러리 설치

```powershell
pip install -r requirement.txt
```

4) 설치 확인(선택)

```powershell
python -c "import pandas, openpyxl, pyautogui, keyboard; print('OK')"
```

5) 모듈 실행 순서

```powershell
# 1. 좌표 확인/보정(필요 시)
python get_pos.py

# 2. 전처리
python data_preprocessing.py

# 3. 자동 기입 매크로
python reduction_apply_typing_macro.py
```

---

## STEP 3. 전처리 결과 확인

### 1) 정상 종료 로그 확인

예시:

```text
📄 입력 CSV: input\data_preprocessing\제98기 일일접수세부내역(20251215-20260222).csv
📄 출력 Excel: output\data_preprocessing\YYYY-MM-DD_output.xlsx
✅ 처리 완료
📊 최종 생성 행 수: N
```

### 2) 결과 파일 확인

- 생성 위치: `output/data_preprocessing/`
- 컬럼 구조:
  - `수강처`
  - `회원번호`
  - `이름`
  - `등록건 수`
  - `강좌`
  - `금액`
  - `감면액`

### 3) 데이터 규칙 확인

- `등록건 수`: 동일 `회원번호 + 이름` 기준으로 1부터 증가
- 정렬: `이름`, `회원번호`, `금액` 순

---

## STEP 4. 매크로 실행 전 화면 세팅

매크로는 **좌표 기반(pyautogui)** 으로 동작합니다.

- Excel/CMS 창 위치를 항상 동일하게 맞춤
- 디스플레이 배율/해상도 변경 금지
- 첫 실행은 반드시 디버그(F8)로 점검

### 엑셀 필터 조건(중요)

- 전처리 결과 엑셀에서 **`등록건 수 = 1`** 만 필터링 후 진행

---

## STEP 5. reduction_apply_typing_macro.py 실행

```powershell
python reduction_apply_typing_macro.py
```

실행 후 동작:

1. PC 환경 선택
   - `1: PC1`
   - `2: PC2`
2. 대기 모드 진입
   - `F8`: 디버그 실행(권장)
   - `F9`: 자동 실행
   - `ESC`: 종료

### 권장 운영 방식

- **1차**: `F8` 디버그로 단계별 클릭/복사/붙여넣기 검증
- **2차**: 이상 없으면 `F9` 자동 실행 후 반복 횟수 입력

---

## STEP 6. 안전 수칙

- 실행 중 마우스/키보드 수동 조작 금지
- 다른 창 클릭 금지(포커스 이탈 시 오입력 위험)
- 비상 중지: 마우스를 화면 좌상단으로 이동(pyautogui FAILSAFE)
- 작업 전 결과 파일 백업 권장

---

## 자주 발생하는 오류와 해결

### 1) `파일이 존재하지 않습니다`

- 원인: 고정 파일명 불일치 또는 위치 오류
- 조치: `input/data_preprocessing/` 경로와 파일명 확인

### 2) 가상환경 활성화 실패

- 원인: PowerShell 실행 정책 제한
- 조치:

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

### 3) 매크로 오클릭/오입력

- 원인: 좌표 불일치(해상도/창 배치 변경)
- 조치: 창 재배치 → `get_pos.py`로 좌표 재측정 → `F8`로 재검증

### 4) 전처리 결과 금액이 예상과 다름

- 점검 포인트:
  - 환불(전액/부분) 반영 여부
  - `감면액` 계산 대상 금액 포함 여부
  - 최신 CSV로 교체했는지
