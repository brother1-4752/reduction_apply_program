# 감면신청서 PDF 생성 상세가이드 (보류 기능)

> 본 문서는 현재 **운영 보류** 상태인 감면신청서 PDF 대량 생성 기능을, 필요 시 즉시 재개할 수 있도록 정리한 실행 가이드입니다.

---

## 1. 기능 개요

- 모듈: `generate_pdf.py`
- 목적: 회원별 감면신청서 PDF 자동 생성
- 출력 위치: `output/generate_pdf/`
- 입력 파일: `input/generate_pdf/results_pdf.xlsx`
- 배경 양식 이미지: `assets/form_bg.png`

현재 운영 프로세스에서는 보류지만, 코드 자체는 실행 가능한 상태입니다.

---

## 2. 실행 전 준비사항

### 2-1. 필수 패키지

프로젝트 루트에서 가상환경 활성화 후 설치:

```powershell
pip install -r requirement.txt
```

`generate_pdf.py`는 `pandas`, `reportlab`, `openpyxl`를 사용합니다.

### 2-2. 입력 파일/경로 확인

- 입력 엑셀 고정 경로: `input/generate_pdf/results_pdf.xlsx`
- 양식 이미지 고정 경로: `assets/form_bg.png`
- 출력 폴더: `output/generate_pdf/` (없으면 자동 생성)

---

## 3. 입력 데이터 규격

`results_pdf.xlsx`에는 최소 아래 컬럼이 필요합니다.

- 회원 식별/기본: `회원번호`, `이름`, `등록건 수`
- 강좌 정보: `강좌`, `금액`, `감면액`
- 개인정보: `주민번호`, `전화번호`, `상세 주소`, `은행명`, `예금주`, `계좌번호`

### 검증 규칙(코드 기준)

- 그룹 기준: `회원번호 + 이름`
- `등록건 수 = 1` 행은 회원당 **정확히 1개**여야 함
- 개인정보 필수값(`주민번호`, `전화번호`, `은행명`, `예금주`, `계좌번호`, `상세 주소`) 누락 시 해당 회원 PDF 생성 스킵
- 회원별 강좌 수가 15개 초과 시 에러 처리 후 스킵

---

## 4. 실행 방법

프로젝트 루트에서 실행:

```powershell
python generate_pdf.py
```

정상 동작 시 콘솔에 파일별 생성 로그가 출력됩니다.

예시:

```text
✅ 생성 완료: 1분기_감면신청서_홍길동_W12345678_1.pdf
```

---

## 5. 생성 로직 요약

- 분기 기본값: `1분기`
- 회원별 강좌를 `등록건 수` 순으로 정렬
- 페이지당 강좌 최대 5개
- 회원당 강좌 최대 15개(최대 3페이지)
- 파일명 규칙:

```text
{분기}_감면신청서_{이름}_{회원번호}_{페이지번호}.pdf
```

예: `1분기_감면신청서_김철수_W10000001_2.pdf`

---

## 6. 테스트 모드 사용법

`generate_pdf.py` 상단 설정값:

- `TEST_MODE = False`
- `TEST_LIMIT_GROUP = 1`
- `TEST_ALL_QUARTERS = True`

### 테스트 권장 패턴

1. `TEST_MODE = True`
2. `TEST_LIMIT_GROUP = 1` (상위 1명만)
3. 필요 시 `TEST_ALL_QUARTERS = True`로 1~4분기 체크박스 위치 확인
4. 결과 확인 후 `TEST_MODE = False`로 복원

---

## 7. 좌표/레이아웃 보정

좌표 정의 파일: `pdf_positions.py`

- 분기 체크 위치(`1분기`~`4분기`)
- 이름/주민번호/회원번호/전화번호
- 주소(괄호 유무 분기 좌표)
- 강좌/금액/감면액 행별 좌표(5줄)
- 은행명/예금주/계좌번호

양식 이미지가 변경되면 좌표 재보정이 필요합니다.

### 좌표 추출 도구

```powershell
python image_coordinate_picker.py
```

- 클릭한 지점의 PDF 좌표(x, y)를 콘솔 출력
- ESC로 종료

---

## 8. 주소 출력 규칙

- 주소에 괄호가 있으면 괄호 기준으로 2줄 분리 출력
- 괄호가 없으면 단일 라인 좌표에 출력

즉, `상세 주소` 값 포맷에 따라 출력 좌표가 자동 분기됩니다.

---

## 9. 자주 발생하는 오류와 해결

### 9-1. 입력 엑셀 파일 없음

- 증상: `input/generate_pdf/results_pdf.xlsx` 관련 파일 오류
- 조치: 경로/파일명 확인 후 재실행

### 9-2. 배경 이미지 없음

- 증상: `assets/form_bg.png` 로드 실패
- 조치: 양식 이미지 파일 존재 여부 확인

### 9-3. 회원별 등록건 수 1행 이상/이하

- 증상: `[ERROR] ... 등록건 수 1 행 n개`
- 조치: 전처리 결과에서 회원별 `등록건 수` 무결성 점검

### 9-4. 개인정보 누락

- 증상: `[ERROR] ... 개인정보 누락: [...]`
- 조치: 필수 입력 항목 보완 후 재생성

### 9-5. 강좌 수 초과

- 증상: `[ERROR] ... 강좌 16개 → 15개 초과`
- 조치: 회원 강좌 데이터 확인(중복/비정상 데이터 정리)
